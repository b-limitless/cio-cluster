## Features
- User account management
- Product management - [Fabric]
- Team members management
- Product customization using 3D Model using Three JS - under processing

## System
- Microservice with Event-driven architecture
- Microfront end architecture

## Development Process
- Test Driven Development: Chai, Mocha, Supertest, Cypress JS
- GitHub Actions and workflows

## Prerequisites
- Node JS, Minikube (currently using on Mac), Docker, Skaffold, Kubernetes

## Getting Started
1. Clone the repository:

    ```bash
    git clone https://github.com/your-username/cio-cluster.git
    ```

2. Navigate to the project directory:

    ```bash
    cd cio-cluster
    ```

3. Install dependencies:

    ```bash
    skaffold dev
    ```
